# table-rwd-demo

$ npm install  
安裝相關套件  

$ gulp default  
scss 存檔檔案自動編譯  
在 scss 檔案編譯完成後，網頁會同步 reload  
可在 http://localhost:3333/ 中找到 RWD Table 的各個 Demo
